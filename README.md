# InputLeap Code - 跨平台键鼠共享系统

一个支持Windows和Mac之间的键鼠共享、剪贴板同步和文件传输的跨平台解决方案。

## 功能特性

- 🖱️ **键鼠共享**: 在多台设备间无缝切换鼠标和键盘控制
- 📋 **剪贴板同步**: 支持文本和图片的实时剪贴板同步
- 📁 **文件传输**: 通过拖拽直接在设备间传输文件
- 🔒 **TLS加密**: 默认启用TLS加密，确保数据传输安全
- 🌐 **设备发现**: 自动发现局域网内的设备
- 🎨 **现代界面**: 基于Electron + React的现代化用户界面

## 系统要求

- Windows 10/11 或 macOS 10.14+
- Node.js 16.0+
- 局域网连接

## 使用方法

### 作为服务器

1. 启动应用
2. 在"连接"选项卡中选择"作为服务器"
3. 设置设备名称和端口（默认24800）
4. 点击"启动服务器"
5. 其他设备可以连接到这台电脑

### 连接到服务器

1. 启动应用
2. 在"连接"选项卡中选择"连接到服务器"
3. 输入服务器IP地址和端口
4. 点击"连接"

### 设备发现

1. 在"设备"选项卡中点击"刷新"
2. 应用会自动搜索局域网内的设备
3. 点击"连接"按钮快速连接

### 文件传输

1. 在"传输"选项卡中选择要发送的文件
2. 选择目标设备
3. 点击"发送文件"

## 安全性

- 所有网络传输默认使用TLS加密
- 支持证书验证（可选）
- 设备间通信使用加密通道
- 文件传输包含完整性校验

## 配置选项

### 网络设置
- 端口配置（默认24800）
- TLS加密开关
- 连接超时设置

### 输入设置
- 鼠标共享开关
- 键盘共享开关
- 鼠标滚轮同步

### 剪贴板设置
- 文本同步
- 图片同步
- 文件同步

### 文件传输设置
- 最大文件大小限制
- 传输超时设置

## 故障排除

### 连接问题
1. 确保设备在同一局域网内
2. 检查防火墙设置
3. 确认端口未被占用

### 权限问题
- macOS: 需要辅助功能权限
- Windows: 可能需要管理员权限

### 性能问题
- 调整鼠标移动阈值
- 减少剪贴板同步频率
- 优化网络设置

## 开发

### 项目结构

```
src/
├── main.js                 # Electron主进程
├── modules/                # 核心模块
│   ├── NetworkManager.js   # 网络通信
│   ├── InputCapture.js     # 输入捕获
│   ├── ClipboardSync.js    # 剪贴板同步
│   ├── FileTransfer.js     # 文件传输
│   └── DeviceDiscovery.js  # 设备发现
└── renderer/               # 渲染进程
    ├── index.html          # 主界面
    └── app.js              # 前端逻辑
```

### 技术栈

- **框架**: Electron + React
- **网络**: Node.js net + TLS
- **设备发现**: mDNS/Bonjour
- **输入处理**: robotjs
- **UI**: Bootstrap 5
